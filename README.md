## 🚀 **What is LiftFire?**

LiftFire is a **next-generation fitness PWA** that gamifies your workout experience through social features, achievement systems, and real-time tracking. Built with cutting-edge web technologies and AI-assisted development, it delivers a native app experience that works offline-first.

### 🎯 **Key Highlights**

- 🎮 **Full Gamification System** - XP, achievements, streaks, and leaderboards
- 💳 **Complete Marketplace** - Stripe-powered payment processing for premium content
- 👥 **Social Fitness Network** - Connect with gym friends and share progress
- 📱 **Native Mobile Experience** - PWA with Android APK support
- 🔒 **Enterprise-Grade Security** - PCI DSS compliant payment processing
- ⚡ **Offline-First Architecture** - Works without internet connection
- 🤖 **AI-Powered Development** - Built using Kiro AI with spec-driven development

## 🚀 **Quick Start**

### **Prerequisites**
- Node.js 18+ 
- npm or yarn
- Android Studio (for mobile development)

### **Installation**

```bash
# Clone the repository
git clone https://github.com/yourusername/liftfire-fitness-pwa.git
cd liftfire-fitness-pwa

# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:5173
```

### **Mobile Development**

```bash
# Build for mobile
npm run build:mobile

# Open in Android Studio
npm run cap:open:android

# Build APK for testing
npm run mobile:build:test
```
